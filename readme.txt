Adds the ability to set a mobile responsive logo so your logo will look great on any device.

== Description ==

Genesis Responsive Header adds a new settings page which allows users to upload multiple logos for various device widths and set a background color if desired.

The requires CSS and Javascript if enabled will be loaded on front end of site to allow for better scaling of image.


This plugin requires the [Genesis Theme Framework](http://designsbynickthegeek.com/go/genesis)

== Installation ==

1. Upload the entire `genesis-responsive-header` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Simple Comment` menu item under `Genesis`
1. Select desired comment options and save settings
